## Get started

To set up this repository, run:

```shell
yarn
```

> **Note**
> This repository uses [MJML](https://documentation.mjml.io/) to generated responsive HTML emails

## Develop

To serve local assets:

```shell
yarn dev:<directory>
```

## Build

To build HTML:

```shell
yarn build:<directory>
```
